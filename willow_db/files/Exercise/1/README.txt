Binary Search Tree Exercise

Print the in-order BST that results when you insert a sequenece of numbers

The input.txt contains the problem instances with the following pattern:
K // The number of problem instances
// line break
// for i in range(0, k):
v0 v1 v2 v3 v4 v5 v6 v7 v8 ... // Line with values to insert in the tree
// line break

The result should be print in the console in the following format:
i // the number of the problem
r0 r1 r2 r3 r4 r5 r6 r7 r8 ... // Line results of BST in order traversal
